# Regjistri i Barnave — Kosovë 2025/2026

Regjistër interaktiv me 4006 produkte medicinale, kërkim, filtra dhe kolona të editueshme.

## Struktura

- `index.html` — dizajni dhe funksionet e faqes
- `data/part-*.js` — databaza e kompresuar e barnave

## Editimi

Hape `index.html` në GitHub dhe kliko ikonën e lapsit për ndryshime në dizajn ose funksione. Të dhënat janë ndarë në skedarët brenda dosjes `data`.

## Shënim mjekësor

Klasat dhe fjalët kyçe janë për kërkim dhe orientim edukativ. Ato nuk zëvendësojnë fletëudhëzimin zyrtar, SmPC-në apo vendimin klinik.
